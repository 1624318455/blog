# 我的博客

一个使用 React + Vite + Ant Design 构建的博客网站。

## 技术栈

- **React 18** - UI 框架
- **Vite 5** - 构建工具
- **Ant Design 5** - UI 组件库
- **TypeScript** - 类型安全
- **React Router 6** - 路由管理

## 功能特性

- 📝 文章列表与详情阅读
- 🏷️ 分类浏览（按标签筛选）
- 🔍 文章搜索
- 📱 响应式设计
- 🎨 优雅的视觉设计

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署

项目已配置 Vercel 部署，可一键部署或通过 GitHub Actions 自动部署。
